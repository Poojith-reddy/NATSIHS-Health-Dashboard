# Tableau Workbook

Place your packaged Tableau workbook here:

- `NATSIHS_Dashboard.twbx` — Packaged workbook (includes data + visualisations)

## How to Export from Tableau

1. Open your workbook in Tableau Desktop
2. Go to **File** → **Export Packaged Workbook...**
3. Save as `NATSIHS_Dashboard.twbx`
4. Place the file in this folder
